My Projects

PacMan

In this project, I created a Pac-Man game using Java Swing, focusing on key concepts like game loops, JFrame, JPanel, and tilemaps. I implemented movement controls for Pac-Man, random ghost movement, collision detection, and scoring. The game resets when Pac-Man collides with a ghost, and I gained hands-on experience in Java game development.

Todo App

Developed a feature-rich Todo app for Android using Kotlin, enabling users to efficiently manage tasks and stay organized. The app allows users to add, edit, and delete tasks with a user-friendly interface and assign priority levels for better organization. It incorporates a Room database for local data persistence, ensuring tasks remain accessible offline. This project highlights proficiency in Android development, lifecycle management, and the integration of key Android libraries.
